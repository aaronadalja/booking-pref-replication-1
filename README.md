# Booking Preferences Replication
data and replication code for 2026 online hotel booking preferences paper

Author: Aaron Adalja

Date: January 2026

The files contained herein allow for replication of the results for:
"Consumer preferences for hotel sustainability attributes in online bookings"
by Sabrina Pan, Aaron Adalja, Megan Nguyen

root folder:

booking_pref_replication.R - This is the R code file that is used to process the data and generate all tables, figures, and statistics for the manuscript.

data folder:

rawdata_low.xlsx - raw survey and experiment data for Economy segment participants 

rawdata_mid.xlsx - raw survey and experiment data for Midscale segment participants 

rawdata_high.xlsx - raw survey and experiment data for Upscale segment participants
